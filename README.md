## Observer

Dark psychology intelligence module built with Next.js 14, Tailwind CSS, and the App Router. The experience delivers cold analysis of manipulation tactics, power mechanics, and defensive awareness with a cinematic interface tuned for Vercel deployment.

### Local Development

```bash
npm install
npm run dev
```

Visit [http://localhost:3000](http://localhost:3000) to explore the interface. Edits inside `src/` trigger fast refresh automatically.

### Production Build

```bash
npm run build
npm run start
```

### Contents

- Scenario decoder with live toggles across executive, political, academic, and competitive arenas.
- Manipulation playbooks, power lifecycle mapping, and influence protocols emphasising awareness and defense.
- Micro-expression lexicon highlighting somatic tells and countermeasures.

Deploy directly with Vercel CLI for a fully static edge-ready build.
